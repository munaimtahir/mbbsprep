Format Guide & Requirements
Required Columns:
REQ Question Text - The full MCQ question
REQ Option A - First answer choice
REQ Option B - Second answer choice
REQ Correct Answer - A, B, C, D, E, or F
REQ Topic - Topic name (will be created if new)
Optional Columns:
OPT Subject - Can use default below
OPT Option C, D, E, F - Additional choices
OPT Difficulty - Easy, Medium, Hard
OPT Tags - Comma-separated
OPT Explanation - Answer explanation
OPT Reference - Source citation
Important Tips:
  • At least 2 options (A, B) are required
  • Correct Answer must match one of your option letters
  • Subject and Topic will be created if they don't exist
  • Tags should be separated by commas
  • File size limit: 10MB
