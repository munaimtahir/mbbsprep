Tips & Information
About Tags

Tags help organize content by theme, making it easier for users to find related materials across subjects.

Best Practices
  • Use clear, specific names for tags
  • Choose distinctive colors for visual recognition
  • Use subtags for more granular organization
  • Specify which resource types can use this tag
Archiving vs. Deletion

Archived tags remain in the database but aren't available for selection. This preserves historical data while keeping the interface clean.

Tips for Effective Tags
  • Use concise, descriptive names
  • Choose distinct colors to help with visual recognition
  • Add relevant subtags to organize related content
  • Be consistent with naming conventions
  • Use tags to help students find relevant content
  • Archive unused tags instead of deleting them
Resource Types

Tags can be assigned to one or more resource types:

  • MCQs - Multiple choice questions in the question bank
  • Videos - Tutorial and lecture video content
  • Notes - Study notes and reference materials
  • All Resources - Apply to all resource types (overrides individual selections)
