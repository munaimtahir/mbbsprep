# Template Map

This document records the functional template ownership model of the repository.

## Public / Student Template Groups

### Authentication templates
Own the signup/login/logout-related UI in scope.

### Dashboard templates
Own the authenticated student landing experience.

### Profile templates
Own profile display and profile edit.

### Quiz templates
Own quiz start, quiz session, and quiz result.

### Resource templates
Own active public resources such as notes, videos, and flashcards.

### Subscription / Payment templates
Own payment instructions, proof upload, and payment status.

## Staff Template Groups

### Dashboard templates
Own staff dashboard experience.

### Subject / Topic / Question management templates
Own current content-admin pages.

### Payment templates
Own payment queue, review page, and history.

### Resource administration templates
Own staff resource listing/management pages in scope.

### Support placeholder templates
Own non-crashing placeholder staff support pages.

### Bulk upload templates
Own preview/confirm/import flow.
