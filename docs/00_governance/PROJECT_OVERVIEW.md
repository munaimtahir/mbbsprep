# Project Overview

## Project Name

**MedPrep** (repository context: `mbbsprep`)

## Project Type

A Django-based medical education platform focused on MBBS exam preparation through MCQs, quizzes, and curated study resources.

## Intended Audience

### Primary users
- MBBS students preparing for professional examinations
- learners using question-bank and quiz-driven revision workflows

### Secondary users
- staff/admin users responsible for content management, user support, and payment verification

## Product Purpose

MedPrep is designed to provide a structured and scalable learning environment where students can:

- register and authenticate
- access a dashboard
- browse question-bank content
- attempt quizzes
- view profile information
- access learning resources such as notes, videos, and flashcards
- submit payment proof where premium access workflows apply

In parallel, staff users can:

- manage academic taxonomy (subjects, topics)
- manage questions/MCQs
- manage resources
- review payment submissions
- manage users and bulk uploads
- access placeholder support administration where implemented

## Current Repository State

The repository has reached a **verified v1 baseline with documented limits**.

This means:

- critical student and staff flows have been stabilized
- major runtime errors are eliminated
- key workflow tests exist
- known limitations are documented

## Strategic Goal

Pause major feature work and establish a full governance/documentation/structure pack so future development remains consistent and safe.
